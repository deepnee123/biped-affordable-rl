# Low-Cost Bipedal Robot via Reinforcement Learning

A personal passion project focused on achieving robust bipedal locomotion using highly accessible, 3D-printed hardware and machine learning control.

## Current State
- [x] Basic physical structure conceptualized
- [/] Setting up simulation and hardware architecture (In Progress)
- [ ] RL Policy training

## System Constraints & Goals
- **Budget:** Under $300
- **Manufacturing:** 3D-printed PLA/PETG parts
- **Control:** Blind locomotion utilizing IMU and joint feedback
